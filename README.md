 SME Business Management System

 Overview
This is a Flask-based web application designed to help small and medium enterprises manage their daily operations.

 Features
- User Authentication (Admin & Staff)
- Product Management
- Sales Recording
- Expense Tracking (Admin only)
- Profit Calculation
- Sales Dashboard with Charts

 Technologies Used
- Python (Flask)
- SQLite
- HTML, CSS, JavaScript
- Chart.js

 Roles
- Admin: Full access (users, expenses)
- Staff: Limited access (sales only)

 Profit Calculation
Profit = Total Sales - Total Expenses
